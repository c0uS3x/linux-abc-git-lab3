# Лабораторная работа №3 - Система контроля версий Git

## Описание
Практическое освоение работы с Git: ветвление, слияние, разрешение конфликтов.

## Вариант: 504978

## Структура проекта
- `run.txt` - основной файл для работы с коммитами
- `diff.txt` - сравнение изменений между коммитами
- `history.txt` - история всех коммитов

## Рабочий процесс
1. Создание и управление ветками
2. Работа с коммитами
3. Разрешение конфликтов при слиянии
4. Создание релизной ветки
